# Homelab
Service configs running in my homelab. Folders are set-up depending on which device the serice will run.

Currently there is only the RPi folder, which will try to follow the following tutorial: https://medium.com/@tim.webster/highly-available-stateless-pi-hole-with-microk8s-9566c5c0f31d and also add two instances of nginx reverse proxy manager for internal certificate handling with a DNS challenge.

The configuration is still a work in progress.
